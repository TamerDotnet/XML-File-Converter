Installed the following packages:
* Microsoft.Extensions.Configuration;
Need to build configuration builder in order to use the Appsettings.json file which contains application settings for file locations

* Microsoft.Extensions.DependencyInjection
Framework needed to inject objects such as FileReader

* Settings for input and output is located within the Appsettings.json file, Application read and generate files within the 
 Processing Files Folder 

* All the settings needed for the Dependency injection done within Setup File (Console Project)

* Unit Testing done using xUnit and Moq (Moq package installed for the project)
